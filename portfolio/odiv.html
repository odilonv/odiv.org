Un site vitrine mais avant tout un apprentissage ?



Odiv.org est un site vitrine personnel qui présente mes différents projets.

J'ai créé ce site en autodidacte en utilisant les dernières technologies web pour offrir une expérience utilisateur immersive.

Le site est entièrement reactif et s'adapte à tous les types d'appareils. J'ai intégré une grande variété de fonctionnalités pour aider les visiteurs à naviguer facilement entre mes différents projets, et j'ai également inclus des fonctionnalités de partage sur les réseaux sociaux.

En somme, odiv.org, c'est un site vitrine personnel qui offre une expérience de navigation agréable et immersive pour découvrir mes différents projets.


Pour plus d'informations, n'hésitez pas à consulter mon GitHub !
